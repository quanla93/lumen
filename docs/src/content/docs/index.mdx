## Build around the operator flow

<CardGrid stagger>
  <Card title="Install the hub" icon="rocket">
    Run Lumen as a single hub service with Docker Compose, a native binary, or inside a Proxmox LXC.
  </Card>
  <Card title="Add agents" icon="setting">
    Create hosts in the UI, copy one-shot tokens, then install agents on Linux, Docker, VMs, or LXCs.
  </Card>
  <Card title="Use the dashboard" icon="laptop">
    Read fleet health, drill into host charts, manage tokens, tune retention, and troubleshoot stale agents.
  </Card>
